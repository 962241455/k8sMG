### K8S发布管理平台
#### 1-1 实现方式
-
- 前端：Vue.js + iview
- 后端：Tornado
- 其他组件：K8S + Jenkins

#### 1-2 项目功能：
- 项目创建，管理，关联应用
- 应用管理，关联代码仓库
- 发布管理，整个项目发布，单个应用发布，发布审核
